# Piet IDE
This is an IDE for the Piet esolang, developped in the Godot game engine.

## Piet
**Piet** is a stack-based esoteric programming language in which programs look like abstract paintings. It uses 20 colors, of which 18 are related cyclically through a lightness cycle and a hue cycle. A single stack is used for data storage, together with some unusual operations.

Piet was invented by *David Morgan-Mar* and is named after geometric abstract art pioneer *Piet Mondrian*.

The original language specifications can be found on David's site [here](https://www.dangermouse.net/esoteric/piet.html).

## The IDE
The current state of the IDE is very basic. It should be able to interpret Piet source "code" files without major issues (no proper testing has been done so far).

It features a stack visualizer, an output field and way to very basically see the Direction Pointer location and state.

This is very much WIP, but it serves as a proof of concept.

## Plans
The plan is that, eventually, one could "easily" code and debug a Piet program with nice visuals to help with flow overview (Piet being a "2D" programming language and all).

Currently being worked on in my free time, for fun.
